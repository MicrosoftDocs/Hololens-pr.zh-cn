# <a name="hololens-2-development-edition-faq"></a>HoloLens 2 开发版常见问题

## <a name="general-questions"></a>一般问题

### <a name="what-do-i-get-with-the-hololens-2-development-edition"></a>如何通过 HoloLens 2 开发版获得哪些功能？

HoloLens 2 开发版在 Azure 信用额度中包括 HoloLens 2、$500、3个月的 Unity Pro 试用版和3个月的 Pixyz 插件试用版。

### <a name="what-is-the-difference-between-hololens-2-and-hololens-2-development-edition"></a>HoloLens 2 和 HoloLens 2 开发版之间的区别是什么？

您只能使用 HoloLens 2 开发版来生成和测试内部企业应用程序，以便与在生产设置中使用 HoloLens 2 (商用版) 配合使用。 在购买开发版的过程中，Microsoft 可能会自行决定是否向您提供在结帐过程中确定的某些权利。 这些权利包括在有限期限内兑换某些 Microsoft 或第三方或第三方服务或工具的特定折扣额度或试用版的权限。 [了解详细信息](hololens2-options.md#learn-about-the-development-edition)

### <a name="can-any-edition-of-hololens-2-be-used-to-develop"></a>是否可以使用任何版本的 HoloLens 2 进行开发？

是的。 任何版本的 HoloLens 2 均可用于开发应用。

### <a name="how-and-will-i-receive-my-500-in-azure-credits-3-month-unity-pro-trial-and-3-month-pixyz-plugin-trial"></a>我如何在 Azure 信用额度、3个月的统一 Pro 试用版和3个月的 Pixyz 插件试用版中收到 $500？

折扣额度或试用版访问将发送到产品发货的2个工作日内在购买时使用的电子邮件。

### <a name="can-i-upgrade-my-hololens-2-device-only-device-to-the-hololens-2-development-edition-to-get-the-trial-credits"></a>我是否可以将我的 HoloLens 2 (设备 "升级" 为仅) 设备升级到 HoloLens 2 开发版，以获得试用信用额度？

不，我们不提供设备传输计划。 不能从任何版本传输到任何其他版本。

### <a name="what-is-unity-pro"></a>什么是 Unity Pro？

Unity Pro 是适用于小型/中型工作室/公司的实时3D 平台。 它适用于混合技术和非技术技能的团队。 [了解详细信息](https://store.unity.com/products/unity-pro)

### <a name="what-is-pixyz-plugin"></a>什么是 Pixyz 插件？

Pixyz 插件读取和转换几乎任何格式，将其优化并置于 Unity 中，并将其阶段用于为实时应用程序提供随时可用的资产。 [了解详细信息](https://unity.com/products/pixyz)

### <a name="when-will-financing-for-the-hololens-2-development-edition-be-available"></a>HoloLens 2 开发版的融资何时可用？

HoloLens 2 开发版的融资将于2021年初推出。

### <a name="who-do-i-contact-with-questions-about-my-hololens-2-development-edition-entitlements"></a>我应该与我的 HoloLens 2 开发版权利相关的问题联系谁？

如果与权利相关的任何问题，请联系你的销售专家，网址为866-425-4709，周一至周五，早晨到6：6，PST。

### <a name="can-my-company-buy-multiple-hololens-2-development-edition-devices"></a>公司是否可以购买多个 HoloLens 2 开发版设备？ 

此设备用于构建混合现实解决方案的开发人员。 我们限制每个客户交易的购买一次。 如果贵公司想要购买多个开发版设备，则需要单独的事务来代表公司购买 HoloLens 2 开发版设备。 

### <a name="can-i-show-my-hololens-2-development-edition-device-to-my-customers"></a>我是否可以向我的客户显示 HoloLens 2 开发版设备？ 

是的，此设备旨在用于设计混合现实解决方案并对其进行测试。

### <a name="where-can-i-buy-the-hololens-2-development-edition"></a>在哪里可以购买 HoloLens 2 开发版？ 

HoloLens 2 开发版仅在 Microsoft Store 上出售。  

### <a name="can-i-return-my-hololens-2-development-edition"></a>我是否可以返回 HoloLens 2 开发版？  

所有销售均为最终状态，且在 HoloLens 2 开发版中没有退款。

### <a name="does-developer-edition-have-different-hardware-requirements-that-differ-from-the-hololens-2"></a>开发人员版有不同于 HoloLens 2 的不同硬件要求吗？

不能，此设备与硬件要求不同。 开发版允许在开发和/或测试应用程序的商业设置中部署协助。 

### <a name="how-can-i-use-mixed-reality-toolkit--unity-for-developing-on-my-hololens-2-development-edition"></a>如何使用混合现实工具包– Unity 在我的 HoloLens 2 开发版上进行开发？

MRTK-Unity 是由 Microsoft 驱动的项目，它提供了一系列组件和功能来加速 Unity 中的跨平台 MR 应用开发。 [了解详细信息](https://microsoft.github.io/MixedRealityToolkit-Unity/README.html)

### <a name="how-can-i-use-mixed-reality-toolkit--unreal-uxt-for-developing-on-my-hololens-2-development-edition"></a>如何使用混合现实工具包– Unreal UXT 在我的 HoloLens 2 开发版上进行开发？

MRTK-Unreal 是要释放的第一个 UX 工具组件，目前仅在 HoloLens 2 上受支持。 组件插件包括代码、蓝图以及用于输入模拟的常见 UX 功能的示例资产、手势交互 Actor、可按按钮组件、操纵器组件和跟踪行为组件。 [了解详细信息](https://microsoft.github.io/MixedReality-UXTools-Unreal/version/public/0.9.x/Docs/WelcomeToUXTools.html)

### <a name="are-os-releases-for-the-hololens-2-development-edition-the-same-as-hololens-2"></a>HoloLens 2 开发版的操作系统版本是否与 HoloLens 2 相同？

是的。

### <a name="can-i-run-released-sideload-or-store-applications-on-my-hololens-2-development-edition-device"></a>我是否能在我的 HoloLens 2 开发版设备上运行 (旁加载发布或存储) 应用程序？

如果你按 HoloLens 2 开发版本的条款和条件遵守，是的，你可以在 Developer Edition 上运行任何发布的应用程序。 我们不限制为此 SKU 使用旁加载或存储的应用程序。

### <a name="where-can-i-learn-more-about-the-hololens-2-development-edition"></a>在哪里可以了解 HoloLens 2 开发版的详细信息？

请访问 HoloLens 2 开发版文档页。 [了解详细信息](https://docs.microsoft.com/hololens/hololens2-options?tabs=device)

### <a name="where-can-i-learn-more-about-developing-mixed-reality-solutions"></a>在哪里可以了解有关如何开发混合现实解决方案的详细信息？

请访问混合现实开发文档的简介。 [了解详细信息](https://docs.microsoft.com/windows/mixed-reality/develop/development?tabs=unityhttps://docs.microsoft.com/en-us/windows/mixed-reality/develop/development?tabs=unity)

### <a name="where-can-i-learn-more-about-designing-mixed-reality-solutions"></a>在哪里可以了解有关设计混合现实解决方案的详细信息？

请访问混合现实设计文档简介。 [了解详细信息](https://docs.microsoft.com/windows/mixed-reality/design/design)
