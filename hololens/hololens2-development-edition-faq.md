# HoloLens 2 开发版常见问题

## 常规

### HoloLens 2 开发版有什么作用？

HoloLens 2 开发版包括 HoloLens 2、Azure 信用额度为 500 美元、3 个月的 Unity Pro 试用版和 3 个月的 Pixyz 插件试用版。

### HoloLens 2 和 HoloLens 2 开发版之间有什么区别？

你只能将 HoloLens 2 开发版用于内部构建和测试企业应用程序，以与生产设置中的 HoloLens 2 (商业) 连接使用。 作为购买开发版的一部分，Microsoft 可能会自行决定提供在结帐过程中确定的某些权利，例如，在有限时段内兑换某些折扣信用或试用某些 Microsoft 或第三方服务或工具的权利。 [了解详细信息](hololens2-options.md?tabs=developeredition#1-learn-about-the-development-edition)

### 可以使用任何版本的 HoloLens 2 进行开发吗？

是。 任何版本的 HoloLens 2 都可用于开发应用。

### 如何以及将会收到 500 美元 Azure 信用额度、3 个月的 Unity Pro 试用版和 3 个月的 Pixyz 插件试用版？

你的折扣信用或试用访问权限将在产品发货后 2 个工作日发送给购买时所使用的电子邮件。

### 能否将我的 HoloLens 2 (设备) HoloLens 2 开发版，才能获得试用信用额度？

不，我们不提供设备传输计划。 无法从任何版本转移到任何其他版本。

### 什么是 Unity Pro？

Unity Pro 是小型/中型电影公司实时 3D 平台。 它专为混合使用技术和非技术技能的团队设计。 [了解详细信息](https://store.unity.com/products/unity-pro)

### 什么是 Pixyz 插件？

Pixyz 插件读取和转换几乎任何格式，优化并将其置于 Unity 中，并阶段化它，为实时应用程序提供随时可用资源。 [了解详细信息](https://unity.com/products/pixyz)

### HoloLens 2 开发版何时提供财务？

HoloLens 2 Development Edition 的财务服务将于 2021 年初提供。

### 如果我有关于 HoloLens 2 Development Edition 权利的问题，我应联系谁？

如果对权利有任何疑问，请联系销售专家，电话为 866-425-4709，周一至周五，上午 6 点到下午 6 点，PST。

### 我的公司能否购买多个 HoloLens 2 Development Edition 设备？ 

此设备适用于生成混合现实解决方案的开发人员。 我们按客户交易限制一次购买。 如果你的公司想要购买多个 Development Edition 设备，它将需要单独交易来代表公司购买 HoloLens 2 Development Edition 设备。 

### 我能否将 HoloLens 2 Development Edition 设备显示给客户？ 

是，此设备用于原型制作和测试混合现实解决方案。

### 在哪里可以购买 HoloLens 2 开发版？ 

HoloLens 2 Development Edition 专门在 Microsoft Store 上出售。  

### 我能否返回 HoloLens 2 开发版？  

所有销售都是最终的，HoloLens 2 Development Edition 上没有退款。

### 开发人员版本是否具有不同的硬件要求与 HoloLens 2 不同？

否，从硬件要求的角度来看，此设备是相同的。 允许部署 Development Edition，帮助在商业环境中开发和/或测试应用程序。 

### 如何使用混合现实Toolkit - Unity 在我的 HoloLens 2 开发版上进行开发？

MRTK-Unity是 Microsoft 推动的项目，提供一组组件和功能，用于加速 Unity 中的跨平台 MR 应用开发。 [了解详细信息](https://microsoft.github.io/MixedRealityToolkit-Unity/README.html)

### 如何在我的 HoloLens 2 Toolkit版本上使用混合现实版 - Unreal UXT 进行开发？

MRTK-Unreal发布的第一个 UX 工具组件，当前仅在 HoloLens 2 上受支持。 组件插件包括用于输入模拟的常用 UX 功能的代码、蓝图和示例资产、手部交互主角、按下按钮组件、操作器组件和关注行为组件。 [了解详细信息](https://microsoft.github.io/MixedReality-UXTools-Unreal/version/public/0.9.x/Docs/WelcomeToUXTools.html)

### HoloLens 2 开发版的操作系统版本是否与 HoloLens 2 相同？

是。

### 我能否在我的 HoloLens 2 (设备上) 或存储已发布的应用程序？

如果你要遵守 HoloLens 2 开发版条款和条件，可以运行 Developer Edition 上的任何已发布应用程序。 我们不限制对此 SKU 使用旁加载或存储的应用程序。

### 在哪里可以了解有关 HoloLens 2 Development Edition 的更多信息？

访问 HoloLens 2 Development Edition 文档页。 [了解详细信息](https://docs.microsoft.com/hololens/hololens2-options?tabs=device)

### 在哪里可以了解有关开发混合现实解决方案的更多信息？

访问混合现实开发文档简介。 [了解详细信息](https://docs.microsoft.com/windows/mixed-reality/develop/development?tabs=unityhttps://docs.microsoft.com/en-us/windows/mixed-reality/develop/development?tabs=unity)

### 在哪里可以了解有关设计混合现实解决方案的更多信息？

访问混合现实设计文档简介。 [了解详细信息](https://docs.microsoft.com/windows/mixed-reality/design/design)

### 在哪里可以获取对 HoloLens Developer Edition 的支持？

在此处 [获取支持。](https://support.serviceshub.microsoft.com/supportforbusiness/create?sapId=96bfb202-bc79-741b-bf7a-774d8b767782)

## Azure

### 如何兑换 $500 Azure 信用额度？

若要兑换 500 美元 Azure 信用额度，请兑换 Azure Pass 促销代码并激活订阅。 [了解详细信息](https://www.microsoftazurepass.com/Home/HowTo?Length=5)

查看 Azure Pass [订阅协议](https://azure.microsoft.com/support/legal/subscription-agreement/?country=US) 并提供 [详细信息](https://azure.microsoft.com/en-us/offers/azure-pass/)。

### 我的 500 美元 Azure 信用将于何时到期？

你的 $500 Azure 信用将在激活 Azure 密钥代码后的 90 天内过期。

### 我可以使用 Azure 信用额度做什么？

借助 Azure 信用额度，你可以开始使用 Microsoft&#39;不断扩展的云服务集构建。 HoloLens 2 开发版中包含的 Azure 信用额度使客户能够快速开始在 Azure 上生成和测试混合现实应用程序。 有关在 Azure 上生成的完整详细信息，请访问 [Azure 资源](https://azure.microsoft.com/en-us/resources/)

### 在哪里可以了解有关 Azure 混合现实服务的信息？

Microsoft 提供了一组混合现实服务，使开发人员能够生成和测试混合现实解决方案。 [了解详细信息](https://docs.microsoft.com/windows/mixed-reality/develop/mixed-reality-cloud-services)
