# <a name="hololens-2-development-edition-faq"></a>HoloLens 2 开发版常见问题解答

## <a name="general-questions"></a>一般问题

### <a name="what-do-i-get-with-the-hololens-2-development-edition"></a>使用开发版HoloLens 2功能？

HoloLens 2 Development Edition 包括 HoloLens 2、500 美元的 Azure 额度、3 个月的 Unity Pro 试用版和 3 个月的 Pixyz 插件试用版。

### <a name="what-is-the-difference-between-hololens-2-and-hololens-2-development-edition"></a>HoloLens 2 开发版HoloLens 2区别是什么？

只能使用 HoloLens 2 Development Edition 来生成和测试内部企业应用程序，以与生产HoloLens 2 (商业版) 一起使用。 作为购买 Development Edition 的一部分，Microsoft 可以自行提供在结帐过程中确定的某些权利。 这些权利包括兑换某些折扣额度或在有限时段内试用某些 Microsoft 或第三方服务或工具的权利。 [了解详细信息](hololens2-options-dev-edition.md)

### <a name="can-any-edition-of-hololens-2-be-used-to-develop"></a>可以使用任何版本的 HoloLens 2进行开发？

是的。 任何版本的 HoloLens 2 都可用于开发应用。

### <a name="how-and-will-i-receive-my-500-in-azure-credits-3-month-unity-pro-trial-and-3-month-pixyz-plugin-trial"></a>如何接收 500 美元的 Azure 额度、3 个月的 Unity Pro 试用版和 3 个月的 Pixyz 插件试用版？

折扣额度或试用访问权限将在产品发货后 2 个工作日发送到购买时使用的电子邮件。

### <a name="can-i-upgrade-my-hololens-2-device-only-device-to-the-hololens-2-development-edition-to-get-the-trial-credits"></a>能否将我的 HoloLens 2 (设备) 升级到 HoloLens 2 Development Edition 以获得试用额度？

否，我们不提供设备传输计划。 无法从任何版本转移到任何其他版本。

### <a name="what-is-unity-pro"></a>什么是 Unity Pro？

Unity Pro 是一个适用于小型/中型工作室/企业实时 3D 平台。 它专为混合了技术和非技术技能的团队设计。 [了解详细信息](https://store.unity.com/products/unity-pro)

### <a name="what-is-pixyz-plugin"></a>什么是 Pixyz 插件？

Pixyz 插件读取和转换几乎任何格式，优化并将其置于 Unity 中，并阶段化它，为实时应用程序提供随时可用资产。 [了解详细信息](https://unity.com/products/pixyz)

### <a name="is-financing-for-the-hololens-2-development-edition--available"></a>HoloLens 2版是否可用？

是的 [，HoloLens 2开发版](https://ms.liftforward.com/hololens?product=hololens-development-edition) 目前仅在 美国。 

### <a name="who-do-i-contact-with-questions-about-my-hololens-2-development-edition-entitlements"></a>我与谁联系，询问有关HoloLens 2版权利的问题？

如果对权利有任何疑问，请通过 866-425-4709、周一到周五、上午 6 点到下午 6 点（PST）联系销售专家。

### <a name="can-my-company-buy-multiple-hololens-2-development-edition-devices"></a>我的公司能否购买多个 HoloLens 2 Development Edition 设备？ 

此设备适用于生成混合现实解决方案的开发人员。 我们限制每个客户交易一次购买。 如果你的公司想要购买多个 Development Edition 设备，则需要单独的交易HoloLens 2代表公司购买 Development Edition 设备。 

### <a name="can-i-show-my-hololens-2-development-edition-device-to-my-customers"></a>我能否向HoloLens 2显示我的开发版设备？ 

是的，此设备旨在用于原型制作和测试混合现实解决方案。

### <a name="where-can-i-buy-the-hololens-2-development-edition"></a>在哪里可以购买 HoloLens 2 开发版？ 

HoloLens 2 Development Edition 专门在 Microsoft Store。  

### <a name="can-i-return-my-hololens-2-development-edition"></a>能否返回我的 HoloLens 2 开发版？

### <a name="does-developer-edition-have-different-hardware-requirements-that-differ-from-the-hololens-2"></a>Developer Edition 的硬件要求是否不同于HoloLens 2？

否，从硬件要求的角度来看，此设备是相同的。 允许部署 Development Edition，帮助在商业环境中开发和/或测试应用程序。

MRTK-Unity 是由 Microsoft 驱动的项目，它提供了一系列组件和功能来加速 Unity 中的跨平台 MR 应用开发。 [了解详细信息](https://microsoft.github.io/MixedRealityToolkit-Unity/README.html)

### <a name="how-can-i-use-mixed-reality-toolkit--unreal-uxt-for-developing-on-my-hololens-2-development-edition"></a>如何使用混合现实工具包 - Unreal UXT 在我的开发版HoloLens 2开发？

MRTK-Unreal是要发布的第一个 UX Tools 组件，目前仅在 HoloLens 2。 组件插件包括代码、蓝图以及用于输入模拟的常见 UX 功能的示例资产、手势交互 Actor、可按按钮组件、操纵器组件和跟踪行为组件。 [了解详细信息](https://microsoft.github.io/MixedReality-UXTools-Unreal/version/public/0.9.x/Docs/WelcomeToUXTools.html)

### <a name="are-os-releases-for-the-hololens-2-development-edition-the-same-as-hololens-2"></a>HoloLens 2 开发版的 OS 版本是否与HoloLens 2？

是的。
