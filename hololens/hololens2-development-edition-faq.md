# HoloLens 开发版常见问题解答

## 常规

### 使用 HoloLens 2 开发版可以获得哪些功能？

HoloLens 2 开发版包括在 Azure 信用中使用 HoloLens 2、$500、3个月的 Unity Pro 试用版和3个月的 Pixyz 插件试用。

### HoloLens 2 和 HoloLens 2 开发版有何区别？ 

您只能使用 HoloLens 2 开发版本，以便在生产设置中使用 HoloLens 2 (商业版) 的内部构建和测试企业应用程序。 作为购买开发版的一部分，Microsoft 可能会自行决定如何在结帐过程中向您提供特定的权利，如兑现特定的折扣信用点数或对某些 Microsoft 或第三方服务或工具的试用访问权限。 [了解详细信息](hololens2-options.md?tabs=developeredition#1-learn-about-the-development-edition)

### 是否可以使用 HoloLens 2 的任何版本进行开发？

是。 任何版本的 HoloLens 2 均可用于开发应用。

### 我如何在 Azure 信用、3个月的统一 Pro 试用和3个月的 Pixyz 插件试用中收到我的 $500？

您的折扣信用点数或试用版访问将发送到在产品发运的2个工作日内购买的电子邮件。

### 我是否可以 "将我的 HoloLens 2 (设备" 仅) 设备升级到 HoloLens 2 开发版本才能获取试用信用点数？

否，我们不提供设备传输计划。 不能从任何版本转移到任何其他版本。 

### 什么是 Unity Pro？ 

Unity Pro 是适用于小型/中型录音室/公司的实时3D 平台。 它专为混合使用技术和非技术技能的团队而设计。 [了解详细信息](https://store.unity.com/products/unity-pro)

### 什么是 Pixyz 插件？ 

Pixyz 插件读取和转换几乎任何格式，优化并将其放入 Unity，并将其阶段提供给实时应用程序的可用资源。 [了解详细信息](https://unity.com/products/pixyz)

### 是否提供 HoloLens 2 开发版的融资？

HoloLens 2 开发版的融资将于2021年初推出。

### 谁可以与我的 HoloLens 2 开发版权利相关问题进行联系？

如果您有任何与权利相关的问题，请联系您的销售专家，6AM 于866-425-4709，周一至周五，星期一至6：6：6： PST。

### 我的公司是否可以购买多个 HoloLens 2 开发版设备？ 

此设备适用于构建混合现实解决方案的开发人员。 我们限制每个客户交易的一个采购。 如果您的公司想要购买多个开发版设备，将需要单独的交易来代表公司购买 HoloLens 2 开发版设备。 

### 是否可以向客户显示 HoloLens 2 开发版设备？ 

是的，此设备旨在用于原型设计和测试混合现实解决方案。

### 在哪里可以购买 HoloLens 2 开发版？ 

HoloLens 2 开发版仅在 Microsoft Store 上销售。  

### 我是否可以退还我的 HoloLens 2 开发版本？  

所有销售均为最终版本，并且 HoloLens 2 开发版没有退款可用。

### 开发人员版本是否与 HoloLens 2 有不同的硬件要求？

否，此设备与硬件要求的透视图中的设备相同。 开发版允许部署到 aiding 中的开发和/或测试应用程序的开发和/或测试，在商业设置中。 

### 如何在 HoloLens 2 开发版上使用混合现实工具包-Unity 进行开发？

MRTK-Unity 是一个 Microsoft 驱动的项目，它提供一组组件和功能，用于加速 Unity 中的跨平台 MR 应用开发。 [了解详细信息](https://microsoft.github.io/MixedRealityToolkit-Unity/README.html)

### 如何使用混合现实工具包-Unreal UXT 在我的 HoloLens 2 开发版上进行开发？

MRTK-Unreal 是要释放的第一个 UX 工具组件，目前仅在 HoloLens 2 上受支持。 组件插件包括针对输入模拟、手动交互参与者、可按下按钮组件、操控程序组件和关注行为组件的常见 UX 功能的代码、设计图和示例资源。 [了解详细信息](https://microsoft.github.io/MixedReality-UXTools-Unreal/version/public/0.9.x/Docs/WelcomeToUXTools.html)

### HoloLens 2 开发版的操作系统版本与 HoloLens 2 是否相同？

是。

### 是否可以在 HoloLens 2 开发版设备上运行 (旁加载发布或存储) 应用程序？

如果你是由 HoloLens 2 开发版本的条款和条件 abiding 的，则可以在开发人员版本上运行任何已发布的应用程序。 我们不会限制此 SKU 的旁加载或存储应用程序的使用。

### 在哪里可以了解有关 HoloLens 2 开发版的详细信息？

请访问 HoloLens 2 开发版文档页面。 [了解详细信息](https://docs.microsoft.com/hololens/hololens2-options?tabs=device)

### 在哪里可以了解有关开发混合现实解决方案的详细信息？

请访问混合现实开发文档简介。 [了解详细信息](https://docs.microsoft.com/windows/mixed-reality/develop/development?tabs=unityhttps://docs.microsoft.com/en-us/windows/mixed-reality/develop/development?tabs=unity)

### 在哪里可以了解有关设计混合现实解决方案的详细信息？

请访问混合现实设计文档简介。 [了解详细信息](https://docs.microsoft.com/windows/mixed-reality/design/design)

### 在哪里可以获得 HoloLens 开发人员版的支持？

[在此处](https://support.serviceshub.microsoft.com/supportforbusiness/create?sapId=96bfb202-bc79-741b-bf7a-774d8b767782)获取支持。

## Azure

### 如何兑换 $500 Azure 点数？

要兑换您的 $500 Azure 信用点数，请兑换您的 Azure Pass 促销代码并激活您的订阅。 [了解详细信息](https://www.microsoftazurepass.com/Home/HowTo?Length=5)

查看 Azure 传递 [订阅协议](https://azure.microsoft.com/support/legal/subscription-agreement/?country=US) 并 [提供详细信息](https://azure.microsoft.com/en-us/offers/azure-pass/)。

### 我的 $500 Azure 点数何时过期？

你的 $500 Azure 点数将在激活 Azure 密钥代码后90天后过期。

### 我可以对我的 Azure 点数执行哪些操作？

通过您的 Azure 信用点数，您可以开始用 Microsoft&#39;s 不断扩大的云服务集进行构建。 HoloLens 2 开发版附带的 Azure 点数是使客户能够快速开始在 Azure 上构建和测试混合现实应用程序。 有关在 Azure 上构建的完整详细信息，请访问 [azure 资源](https://azure.microsoft.com/en-us/resources/)

### 在哪里可以了解有关 Azure mixed reality 服务的详细信息？

Microsoft 提供了一组混合现实服务，使开发人员能够构建和测试混合现实解决方案。 [了解详细信息](https://docs.microsoft.com/windows/mixed-reality/develop/mixed-reality-cloud-services)
