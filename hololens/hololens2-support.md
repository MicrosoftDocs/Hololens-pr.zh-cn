## <a name="azure-mixed-reality-services-support"></a>Azure 混合现实服务支持

### <a name="open-an-azure-portal-technical-support-ticket"></a>打开 Azure 门户技术支持票证

若要在 Azure 空间锚、Azure 对象锚或 Azure 远程呈现的 Azure 门户上打开技术支持票证：

1. 打开 [Azure 门户](https://azure.microsoft.com/account/)后，从顶部菜单栏选择“帮助”图标，然后选择“帮助 + 支持”按钮。

   ![Azure 门户帮助与支持](../hololens/images/azure-help-support.png)

2. 打开“帮助 + 支持”页后，选择“+ 新建支持请求”。

   ![Azure 门户的“新建支持请求”](../hololens/images/azure-help-support2.png)

3. 填写完支持票证字段后：

    + 问题类型：技术
    + 服务： Azure 空间锚，Azure 远程渲染或 Azure 对象锚点

    ![Azure 门户的支持票证字段](../hololens/images/azure-help-support3.png)

    1. 单击页面底部的 " **下一步：解决方案** " 以添加更多详细信息，告诉我们有关此问题的详细信息。

    2. 根据您的首选方式，尽可能准确、详细地填写表单。

    3. 查看之前部分填写的所有信息，然后提交支持请求。
