## <span data-ttu-id="147de-107">使用“开始”菜单</span><span class="sxs-lookup"><span data-stu-id="147de-107">Use the Start menu</span></span>

<span data-ttu-id="147de-108">可通过 HoloLens 上的“开始”菜单打开应用、查看重要状态信息和访问工具（如照相机）。</span><span class="sxs-lookup"><span data-stu-id="147de-108">The Start menu on HoloLens is where you'll open apps, see important status info, and access tools like the camera.</span></span>

<span data-ttu-id="147de-109">无论位于 HoloLens 中的任何位置，始终可以使用**开始手势**打开“开始”菜单。</span><span class="sxs-lookup"><span data-stu-id="147de-109">Wherever you are in HoloLens, you can always open the Start menu by using **Start gesture**.</span></span>  <span data-ttu-id="147de-110">在 HoloLens（第一代）上，开始手势为[开花手势](https://support.microsoft.com/help/12644/hololens-use-gestures)。</span><span class="sxs-lookup"><span data-stu-id="147de-110">On HoloLens (1st gen) the Start gesture is [bloom](https://support.microsoft.com/help/12644/hololens-use-gestures).</span></span> <span data-ttu-id="147de-111">在 HoloLens 2 上，[开始手势](hololens2-basic-usage.md#start-gesture)为点击手腕上显示的“开始”图标。</span><span class="sxs-lookup"><span data-stu-id="147de-111">On HoloLens 2, the [Start gesture](hololens2-basic-usage.md#start-gesture) is to tap the Start icon that appears on your wrist.</span></span>  <span data-ttu-id="147de-112">也可以通过说“转到开始菜单”，使用语音打开“开始”菜单。</span><span class="sxs-lookup"><span data-stu-id="147de-112">You can also open the Start menu using your voice by saying "Go to Start".</span></span>

> [!TIP]
> <span data-ttu-id="147de-113">当“开始”菜单处于打开状态时，使用开始手势可将其关闭，或者查看“开始”菜单，然后说“关闭”。</span><span class="sxs-lookup"><span data-stu-id="147de-113">When the Start menu is open, use the Start gesture to close it, or look at the Start menu and say "Close".</span></span>

<span data-ttu-id="147de-114">在“开始”菜单的顶部，你将看到 WLAN、电池、音量和时钟的状态指示器。</span><span class="sxs-lookup"><span data-stu-id="147de-114">At the top of the Start menu, you'll see status indicators for Wi-Fi, battery, volume, and a clock.</span></span> <span data-ttu-id="147de-115">在 HoloLens 2 上，还有一个侦听指示器，它显示设备是否已启用语音，以及是否正在侦听语音命令。</span><span class="sxs-lookup"><span data-stu-id="147de-115">On HoloLens 2 there is also a listening indicator that shows whether the device is speech enabled and is listening for voice commands.</span></span> <span data-ttu-id="147de-116">在底部，你会看到**照片**和**视频**按钮，可以用它们拍照和录制视频。</span><span class="sxs-lookup"><span data-stu-id="147de-116">At the bottom you'll find the **Photo** and **Video** buttons which allow you to take photos and video recordings.</span></span>  <span data-ttu-id="147de-117">还有一个**连接**按钮，通过它你可使用 Miracast 将所看到的内容投影到另一台设备上。</span><span class="sxs-lookup"><span data-stu-id="147de-117">There is also a **Connect** button that allows you to project what you see to another device using Miracast.</span></span>

### <span data-ttu-id="147de-118">在“开始”菜单上查找应用</span><span class="sxs-lookup"><span data-stu-id="147de-118">Find apps on Start menu</span></span>

<span data-ttu-id="147de-119">“开始”菜单中有一个**已固定应用**列表和一个**所有应用**列表。</span><span class="sxs-lookup"><span data-stu-id="147de-119">The Start menu has a **Pinned apps** list and an **All apps** list.</span></span>

- <span data-ttu-id="147de-120">**已固定应用**列表显示已固定的应用。</span><span class="sxs-lookup"><span data-stu-id="147de-120">The **Pinned apps** list shows apps that have been pinned.</span></span> <span data-ttu-id="147de-121">你可以使用**选择并按住**应用磁贴时显示的上下文菜单从**已固定应用**列表中添加和删除应用。</span><span class="sxs-lookup"><span data-stu-id="147de-121">You can add and remove apps from the **Pinned apps** list using the context menu that appears when you **select and hold** on an app tile.</span></span>

- <span data-ttu-id="147de-122">**所有应用**列表显示设备上安装的所有应用。</span><span class="sxs-lookup"><span data-stu-id="147de-122">The **All apps** list shows all apps that are installed on the device.</span></span>  <span data-ttu-id="147de-123">选择**开始**菜单右侧的**所有应用**按钮可访问该列表。</span><span class="sxs-lookup"><span data-stu-id="147de-123">Select the **All apps** button on the right side of the **Start** menu to get to the list.</span></span>

<span data-ttu-id="147de-124">在这两个应用列表上，使用“开始”菜单右侧的**上一页**和**下一页**按钮可逐页浏览列表中的所有应用。</span><span class="sxs-lookup"><span data-stu-id="147de-124">On both app lists, use the **Page up** and **Page down** buttons on the right side of the Start menu to page through all the apps in the list.</span></span>  <span data-ttu-id="147de-125">这两个应用列表将自动打开到上次在设备会话期间使用的页面。</span><span class="sxs-lookup"><span data-stu-id="147de-125">Both app lists will automatically open to the page that was last used during a device session.</span></span>

> [!TIP]
> <span data-ttu-id="147de-126">在 HoloLens 2 上，你可以使用食指直接滚动浏览应用列表。</span><span class="sxs-lookup"><span data-stu-id="147de-126">On HoloLens 2, you can directly scroll the app lists using your index finger.</span></span> <span data-ttu-id="147de-127">只需用指尖触摸列表，然后向上或向下拖动即可。</span><span class="sxs-lookup"><span data-stu-id="147de-127">Just touch the list with your finger tip and drag upwards or downwards.</span></span>

### <span data-ttu-id="147de-128">从“开始”菜单打开应用</span><span class="sxs-lookup"><span data-stu-id="147de-128">Open apps from Start menu</span></span>

<span data-ttu-id="147de-129">若要从“开始”菜单打开应用，只需**选择\*\*\*\*应用磁贴**即可。</span><span class="sxs-lookup"><span data-stu-id="147de-129">To open an app from the Start menu, simply **select** an **app tile**.</span></span> <span data-ttu-id="147de-130">你也可以通过说应用的名称将其打开。</span><span class="sxs-lookup"><span data-stu-id="147de-130">You can also say the name of an app to open it.</span></span>

<span data-ttu-id="147de-131">从“开始”菜单打开应用时，将发生以下情况之一，具体取决于应用的设计方式：</span><span class="sxs-lookup"><span data-stu-id="147de-131">When you open an app from the Start menu, one of the following will happen, depending on how the app is designed:</span></span>

- <span data-ttu-id="147de-132">放置一个**应用窗口**。</span><span class="sxs-lookup"><span data-stu-id="147de-132">An **app window** is placed.</span></span> <span data-ttu-id="147de-133">然后，在该窗口中加载应用，你可以像使用触摸屏一样使用它。</span><span class="sxs-lookup"><span data-stu-id="147de-133">The app is then loaded in the window and you can use it like a touch screen.</span></span>
- <span data-ttu-id="147de-134">将放置一个沉浸式应用的 **3D 应用启动器**。</span><span class="sxs-lookup"><span data-stu-id="147de-134">A **3D app launcher** for an immersive app is placed.</span></span> <span data-ttu-id="147de-135">然后，你需要**选择**该启动器以打开沉浸式应用。</span><span class="sxs-lookup"><span data-stu-id="147de-135">You need to then **select** the launcher to open the immersive app.</span></span>

> [!NOTE]

## <span data-ttu-id="147de-143">在 HoloLens 上使用应用</span><span class="sxs-lookup"><span data-stu-id="147de-143">Using apps on HoloLens</span></span>

<span data-ttu-id="147de-144">HoloLens 上的应用可以使用应用窗口视图或沉浸式视图。</span><span class="sxs-lookup"><span data-stu-id="147de-144">Apps on HoloLens can use app window view or immersive view.</span></span> <span data-ttu-id="147de-145">使用应用窗口视图时，应用只在窗口内显示其内容。</span><span class="sxs-lookup"><span data-stu-id="147de-145">With app window view the app simply shows its content inside a window.</span></span> <span data-ttu-id="147de-146">使用沉浸式视图时，应用会让你退出混合现实空间，以便它可以在你四周的物理环境中显示其内容。</span><span class="sxs-lookup"><span data-stu-id="147de-146">With immersive view an app takes you away from mixed realty home where it can then display its content in the physical environment all around you.</span></span> <span data-ttu-id="147de-147">应用还可以选择同时使用这两个视图。</span><span class="sxs-lookup"><span data-stu-id="147de-147">Apps can also choose to use both views.</span></span>
