# <a name="hololens-1st-gen-hardware"></a>HoloLens（第 1 代）硬件

![Microsoft HoloLens (第一代) ](images/see-through-400px.jpg)

Microsoft HoloLens (第一代) 是世界上第一台完全未保护的全息计算机。 HoloLens通过全息体验重新定义个人计算，以新的方式实现能力。 HoloLens将先进的光学和传感器混合在一起，以提供固定到你周围的现实世界的 3D 全息影像。

## <a name="hololens-components"></a>HoloLens组件

![显示框内容的图像](images/hololens-box-contents.png)

你的HoloLens包括以下功能：

- **Visor**。 包含HoloLens和显示器。 当你正在累累时，可以向上旋转视HoloLens。
- **头带**。 若要打开HoloLens，请使用调整滚轮展开头带。 在HoloLens后，将调整滚轮旋转到头带舒适。
- **亮度按钮**。 当你在设备HoloLens时，亮度按钮位于设备左侧。
- **音量按钮**。 当你正在不HoloLens时，音量按钮位于设备右侧。
- **设备设备**。 当你拾取、打开或HoloLens时，请始终按设备手抓取或按住它。

### <a name="power-supply-details"></a>电源详细信息

设备随设备一起提供电源和 USB 电缆是支持的最佳充电机制。 电源是一个 18W 的电源。  它在 2A 时提供 9V。

充电速率和速度可能因设备运行环境而异。

若要在设备打开时保持/提升内部电池电量百分比，必须至少连接到 15W 的电池电量。

## <a name="care-and-cleaning"></a>注意和清理

请仔细HoloLens你的操作。 若要提升、携带和调整HoloLens，请使用设备设备 &mdash; ，而不是开销开销。 使视器没有脏物和指纹，并避免触摸它。 重复清理可能会损坏视区，因此请尝试保持设备干净。

请勿在应用程序上使用任何HoloLens，并且不要将其放在水或直接将水应用于它。

若要清理视口，请用一只头毛笔刷或灯泡样式的镜头光头笔来清除任何脏物。 使用少量提取的水来微fiber 中微fiber，然后使用它以循环运动擦除视线孔。

若要清理设备的其余部分（包括头带和设备手部），请使用一个含 <9>子和水的无 lint <9>。 请让HoloLens完全干，然后再再次使用它。

![显示如何清理视区的图像](images/hololens-cleaning-visor.png)

## <a name="finding-the-serial-number"></a>查找序列号

第一代HoloLens (设备的序列号) 右侧镜头下方。
1. 将设备放在你前面，就像要在你的头上戴它一样。 将设备向前和垂直。
1. 在镜头仍向前移动时，将设备向下翻转，以便看到底面。
1. 右侧镜头现在位于左侧，在带区上的镜头下打印的是序列号。 

也可通过连接的电脑找到序列号：

1. 插入设备
1. 在文件 **资源管理器中导航** 到此电脑
1. 右键 **单击并选择设备** HoloLens属性
1. 这将显示设备的序列号。 如下面的屏幕截图所示。
