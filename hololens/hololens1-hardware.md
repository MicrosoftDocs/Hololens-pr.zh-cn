# <a name="hololens-1st-gen-hardware"></a>HoloLens（第 1 代）硬件

![Microsoft HoloLens (第一代) ](images/see-through-400px.jpg)

Microsoft HoloLens (第一代) 是世界上第一台完全未保护的全息计算机。 HoloLens通过全息体验重新定义个人计算，以新的方式实现能力。 HoloLens将先进的光学和传感器混合在一起，以提供固定到你周围的现实世界的 3D 全息影像。

## <a name="hololens-components"></a>HoloLens 组件

![显示框内容的图像](images/hololens-box-contents.png)

你的HoloLens包括以下功能：

- 面罩。 包含 HoloLens 传感器和显示器。 当你正在累累时，可以向上旋转视HoloLens。
- 头带。 若要打开HoloLens，请使用调整滚轮展开头带。 在HoloLens后，将调整滚轮旋转到头带舒适。
- 亮度按钮。 当你在设备HoloLens时，亮度按钮位于设备左侧。
- 音量按钮。 当你正在不HoloLens时，音量按钮位于设备右侧。
- **设备设备**。 当你拾取、打开或HoloLens时，请始终按设备手抓取或按住它。

### <a name="power-supply-details"></a>电源详细信息

设备附带的电源和 USB 线缆是支持的最佳充电装置。 电源是一个 18W 充电器。  它在 2A 时提供 9V。

充电率和速度可能因设备运行环境而异。

为了在设备处于打开状态时保持/提高内部电池充电百分比，至少必须将其连接到 15W 充电器。

## <a name="care-and-cleaning"></a>注意和清理

仔细处理 HoloLens。 若要抬起、携带和调整 HoloLens，请使用设备臂 &mdash; 而不是系统开销。 若要使面板无灰尘和指纹，请避免触摸。 重复清理可能会损坏面板，因此请尝试保持设备整洁。

不要在您的 HoloLens 上使用任何清除程序或 solvents，不要在水中 submerge 它或直接对其应用水。

若要清洗面板，请使用 camel 或 goat 的镜头画笔或灯泡样式的镜头风机来去除灰尘。 使用少量经过加工的水轻轻 moisten microfiber 抹布，然后使用它在循环动作中轻轻擦拭面板。

若要清理设备的其余部分，包括 headband 和设备臂，请使用带有轻度 soap 和水的不起毛的软布 moistened。 再次使用之前，让你的 HoloLens 晾干。

![显示如何清理面板的图像](images/hololens-cleaning-visor.png)

## <a name="finding-the-serial-number"></a>查找序列号

HoloLens (第一代) 设备的序列号打印在右侧镜头下。
1. 将设备放在您前面，就像您在头戴上一样。 与正面和直立的设备配合工作。
1. 当镜头仍向前旋转时，翻转设备，以便您可以看到下面。
1. 右侧的镜头现在会显示在左侧，在带区的 "镜头" 下打印为序列号。 

也可以通过已连接的电脑找到序列号：

1. 插上设备
1. 在文件资源管理器中导航到此电脑
1. 右键单击并选择 HoloLens 设备的“属性”
1. 这会显示设备的序列号。 如下图所示。
