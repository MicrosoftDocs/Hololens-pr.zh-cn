# <a name="hololens-1st-gen-hardware"></a>HoloLens (第一代) 硬件

![Microsoft HoloLens (第一代) ](images/see-through-400px.jpg)

Microsoft HoloLens (第一代) 是世界上第一个完全 untethered 的全息计算机。 HoloLens 通过全息体验重新定义个人计算，以便以全新方式为你。 HoloLens 混合了先进的器件和传感器，可提供固定到现实世界的3D 全息影像。

## <a name="hololens-components"></a>HoloLens 组件

![显示框内容的图像](images/hololens-box-contents.png)

HoloLens 包含以下功能：

- **面板**。 包含 HoloLens 传感器，并显示。 在戴上，你可以将面板向上旋转。
- **Headband**。 若要将 HoloLens 置于上，请使用调整轮展开 headband。 在设置了 HoloLens 后，将调整滑轮拧紧，直至 headband。
- **亮度按钮**。 在戴上 HoloLens 时，亮度按钮位于设备的左侧。
- **音量按钮**。 在戴上 HoloLens 时，音量按钮位于设备的右侧。
- **设备扶手**。 当你选择、投入或取出你的 HoloLens 时，始终抓住或抓住设备臂。

## <a name="in-the-box"></a>框中

> [!VIDEO https://www.microsoft.com/videoplayer/embed/c7ceb904-9d5f-4194-9e10-e8a949dbad7d]  

HoloLens 框包含以下项：

- **鼻子 pad**。 选择适合你的鼻子形状的鼻子 pad，并适应你的 eyewear。
- **高架**。 当你在四处移动时戴上了 HoloLens 时，请使用高架带来帮助保持设备的位置。 此外，如果你在短期内戴上短期，使用系统开销可能会使设备更加舒适地磨损。
- **微 USB 电缆**。 使用微型 USB 电缆将您的 HoloLens 连接到电源进行充电，或使用它将您的 HoloLens 连接到您的计算机。
- **电源**。 插入电源插座。
- **Microfiber 布**。 使用软布清洁 HoloLens 面板。

>[!TIP]
>[Clicker](hololens1-clicker.md)附带了 HoloLens (第一代) ，位于单独的框中。

### <a name="power-supply-details"></a>电源详细信息

设备附带的电源和 USB 电缆是用于收费的最受支持的机制。 电源是18W 充电器。  它在2A 提供9V。

充电速率和速度可能因设备运行的环境而异。

若要在设备处于开启状态时保持/提前电池电量百分比，则必须将其连接到15W 充电器。

## <a name="care-and-cleaning"></a>注意和清理

仔细处理 HoloLens。 若要抬起、携带和调整 HoloLens，请使用设备臂 &mdash; 而不是系统开销。 若要使面板无灰尘和指纹，请避免触摸。 重复清理可能会损坏面板，因此请尝试保持设备整洁。

请勿在你的 HoloLens 上使用任何 solvents 或 submerge，也不要在水中将其或直接应用。

若要清洗面板，请使用 camel 或 goat 的镜头画笔或灯泡样式的镜头风机来去除灰尘。 使用少量经过加工的水轻轻 moisten microfiber 抹布，然后使用它在循环动作中轻轻擦拭面板。

若要清理设备的其余部分，包括 headband 和设备臂，请使用带有轻度 soap 和水的不起毛的软布 moistened。 在再次使用之前，让你的 HoloLens 晾干完全。

![显示如何清理面板的图像](images/hololens-cleaning-visor.png)

## <a name="finding-the-serial-number"></a>查找序列号

HoloLens (第一代) 设备的序列号打印在右侧镜头下。
1. 将设备放在您前面，就像您在头戴上一样。 与正面和直立的设备配合工作。
1. 当镜头仍向前旋转时，翻转设备，以便您可以看到下面。
1. 右侧的镜头现在会显示在左侧，在带区的 "镜头" 下打印为序列号。 

还可以通过连接的 PC 找到序列号：

1. 插入设备
1. 在文件资源管理器中导航到 **此电脑**
1. 右键单击并选择 "HoloLens 设备的 **属性** "
1. 这会显示设备的序列号。 如下图所示。
