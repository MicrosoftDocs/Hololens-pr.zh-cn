# [HoloLens 2 开发版](#tab/developeredition)

## 1. 了解开发版本

> [!NOTE] 
> 开发版不允许与商业分布式解决方案一起部署。 开发版允许部署到 aiding 中的开发和/或测试应用程序的开发和/或测试，在商业设置中。  

HoloLens 2 开发版包括在 Azure 信用中使用 HoloLens 2、$500、3个月的 Unity Pro 试用版和3个月的 PiXYZ 插件试用：

| 捆绑功能 | 描述 |
|---|---|
|  [Unity](https://unity.com/) | 一个实时的3D 开发平台。   |
|  [适用于 Unity 的 Pixyz 插件](https://www.pixyz-software.com/plugin/) | 用于将工程 &amp; CAD 世界与开发世界连接起来的桥。   |
| [适用于 Azure 服务的 Azure 信用 $500](https://azure.microsoft.com/resources/) | HoloLens 2 开发版附带的 Azure 点数是快速开始在 Azure 上构建和测试混合现实应用程序。 |

每个 HoloLens 2 版本都具有相同的硬件和 [包装盒中](../hololens2-hardware.md#hololens-components) 的相同内容，并且随附有相同的 [预安装软件](../hololens2-hardware.md#pre-installed-software)。

## 2. 兑换您的信用点数

### Unity
您将收到一个唯一的代码。 有关兑换 Unity Pro 试用版的说明： 
1. 访问 [id.unity.com](http://id.unity.com/) 
1. 用您的 Unity ID 登录或创建一个
1. 单击 "**兑换产品代码**" 
1. 输入产品代码，然后单击 "**兑换产品代码**"

有关 [如何激活 Unity 许可证](https://support.unity3d.com/hc/articles/211438683-How-do-I-activate-my-license-)的更多详细信息。 联系 [Unity 支持](https://support.unity3d.com/hc) 人员以获取所需帮助。  

### Pixyz
您将收到一个唯一的代码。 兑换 Pixyz 插件试用的说明： 
1. 访问 [id.unity.com](http://id.unity.com/)
1. 单击 "**兑换产品代码**"
1. 输入你的 Pixyz 插件产品代码，然后单击 "**兑换产品代码**"

有关如何[兑换你的 Pixyz 许可证](https://www.pixyz-software.com/documentations/html/2020.1/review/TrialLicense.html)的更多详细信息。 如有必要，请联系 [Pixyz 支持人员](https://www.pixyz-software.com/support/) 以获取帮助。 

### Azure
下面是一些 [适用于混合现实的常见 Azure 产品](https://azure.microsoft.com/topic/mixed-reality/) ，您可能希望兑换您的 Azure 点数。
如何兑换 Azure 信用点数：要兑换您的 $500 Azure 信用点数，请兑换您的 Azure Pass 促销代码并激活您的订阅。 [了解详情。](../hololens2-development-edition-faq.md#how-can-i-redeem-my-500-azure-credit)

- [了解 Azure 空间锚点可如何帮助你构建映射、共享和保留3D 内容的应用](https://azure.microsoft.com/services/spatial-anchors/)
- [了解 Azure 远程呈现如何呈现高质量的交互式3D 内容，以及如何实时将其流式传输到你的设备](https://azure.microsoft.com/services/remote-rendering/)
