# [HoloLens 2 Development Edition](#tab/developeredition)

## 1. 了解开发版本

> [!NOTE]
> 不允许使用商业分发的解决方案部署 Development Edition。 允许部署 Development Edition，帮助在商业环境中开发和/或测试应用程序。  

HoloLens 2 开发版包括 HoloLens 2、Azure 信用额度为 500 美元、3 个月的 Unity 专业试用版和 3 个月的 PiXYZ 插件试用版：

| 捆绑功能 | 说明 |
|---|---|
|  [Unity](https://unity.com/) | 实时 3D 开发平台。   |
|  [适用于 Unity 的 Pixyz 插件](https://www.pixyz-software.com/plugin/) | 连接工程 &amp; CAD 世界与开发世界之间的桥梁。   |
| [Azure 服务的 Azure 信用额度为 500 美元](https://azure.microsoft.com/resources/) | HoloLens 2 开发版中包含的 Azure 信用可快速开始在 Azure 上生成和测试混合现实应用程序。 |

每个 HoloLens 2 版本具有相同的硬件，框中 [](../hololens2-hardware.md#hololens-components)的内容相同，并附带相同的[预安装软件](../hololens2-hardware.md#pre-installed-software)。

## 2. 兑换你的信用

### Unity
你已收到唯一的代码。 兑换 Unity Pro 试用版的路线：
1. 请访问 [id.unity.com](http://id.unity.com/)
1. 使用 Unity ID 登录或创建一个
1. 单击 **"兑换产品代码"**
1. 输入产品代码，然后单击 **"兑换产品代码"**

有关如何 [激活 Unity 许可证的更多详细信息](https://support.unity3d.com/hc/articles/211438683-How-do-I-activate-my-license-)。 根据需要 [联系 Unity](https://support.unity3d.com/hc) 支持人员寻求帮助。  

### Pixyz
你已收到唯一的代码。 兑换 Pixyz 插件试用版的说明：
1. 请访问 [id.unity.com](http://id.unity.com/)
1. 单击 **"兑换产品代码"**
1. 输入 Pixyz 插件产品代码，然后单击 **"兑换产品代码"**

有关如何兑换 [Pixyz 许可证的更多详细信息。](https://www.pixyz-software.com/documentations/html/2020.1/review/TrialLicense.html) 根据需要 [联系 Pixyz](https://www.pixyz-software.com/support/) 支持人员寻求帮助。

### Azure
以下是你可能希望兑换 Azure 信用的 [混合](https://azure.microsoft.com/topic/mixed-reality/) 现实中的一些常见 Azure 产品。
如何兑换 Azure 信用：要兑换 500 美元 Azure 信用额度，请兑换你的 Azure Pass 促销代码并激活你的订阅。 [了解详情。](../hololens2-development-edition-faq.md#how-can-i-redeem-my-500-azure-credit)

- [了解 Azure 空间定位标记如何帮助你构建映射、共享和保留 3D 内容的应用](https://azure.microsoft.com/services/spatial-anchors/)
- [了解 Azure 远程呈现如何呈现高质量的交互式 3D 内容，以及如何实时将内容流式处理到设备](https://azure.microsoft.com/services/remote-rendering/)
